### 💻 在本機端運行

**前置準備**：您的電腦裡需要有 `Node.js` 喔！

1.  **安裝相依套件**：
    打開您的終端機，輸入 `npm install` 把所有需要的魔法小工具都裝起來！
2.  **設定您的 API 金鑰**：
    在 `.env.local` 檔案中，將 `GEMINI_API_KEY` 設定為您自己的 Gemini API 金鑰。這就像是專屬於您的秘密通關密語！🔑
3.  **啟動 App**：
    輸入 `npm run dev`，然後就可以在瀏覽器中看到您的心血結晶囉！🎉
